OpenAlex -> Azure Blob

1. Create Azure VM
2. Connect to Azure VM
3. Install rclone:
	sudo -v ; curl https://rclone.org/install.sh | sudo bash
4. Configure rclone:
	rclone config create source s3 provider AWS env_auth false region us-east-1
	rclone config create dest azureblob sas_url "https://aegisscholardata.blob.core.windows.net/?sv=2024-11-04&ss=bfqt&srt=sco&sp=rwdlacupiytfx&se=2026-05-30T08:31:15Z&st=2026-02-12T01:16:15Z&spr=https&sig=pU5C5a%2B%2BxE1zvMMv3vjUqjlJXC9dMgpsVyM0V%2FfuEIo%3D"
5. Run rclone (start the transfer)
	rclone copy source:openalex/data dest:raw/openalex --progress --transfers=32
